= 查看集群資訊和合約詳情
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以直接從NetApp Console查看容量資訊以及授權資訊。



== 查看容量資訊

當您從「系統」頁面選擇本機ONTAP叢集時，「系統詳細資料」頁面將顯示叢集的容量信息，包括總容量、已使用容量和可用容量。



== 查看許可證和合約詳情

您可以從「訂閱和授權」頁面查看每個本地ONTAP集群的合約詳情。
