= 從NetApp控制台中刪除本機ONTAP系統
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
如果您不再想從控制台管理本機ONTAP系統，請將其刪除。

.所需的控制台角色：
存儲管理員。link:https://docs.netapp.com/us-en/console-setup-admin/reference-iam-predefined-roles.html["了解NetApp控制台存取角色。"^]

刪除系統不會影響ONTAP叢集。您可以隨時從控制台重新發現它。

.步驟
. 從導覽選單中，選擇*儲存>管理*。
. 在「*系統*」頁面上選擇本機ONTAP系統。
. 選擇選單圖示並選擇*從工作區中刪除*。
. 選擇*刪除*進行確認。
